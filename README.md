# Instructor Effectiveness Modeling
for dataset link : https://drive.google.com/file/d/1GUDfEQ9Kc-eoSGCxNgjgaF_T-1O4yMzc/view?usp=sharing



This project analyzes instructor effectiveness in an EdTech context by defining a custom performance metric weighted across learner outcomes, engagement, and feedback. It utilizes a Random Forest model to classify instructors into effectiveness tiers (Low, Medium, High) based on aggregated batch-level data. The analysis emphasizes structured reasoning, feature importance, and the ethical considerations of using ML for performance evaluation.
